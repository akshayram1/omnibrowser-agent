Local-first browser AI automation library

OmniBrowser Agent helps you run page observation, planning, and execution flows directly in the browser.

NPM Package GitHub Repo

Docs

Quick start:

npm install @akshayram1/omnibrowser-agent
import { createBrowserAgent } from "@akshayram1/omnibrowser-agent";

const agent = createBrowserAgent(
  {
    goal: "Open CRM and find customer John Smith",
    mode: "human-approved"
  },
  {
    onStep: (result) => console.log(result.message)
  }
);

await agent.start();

Full project docs are available in this repository under Architecture, Embedding, and Roadmap.
