Installation

npm install @akshayram1/omnibrowser-agent

Quick Start

import { createBrowserAgent } from "@akshayram1/omnibrowser-agent";

const agent = createBrowserAgent(
  {
    goal: "Open CRM and find customer John Smith",
    mode: "human-approved"
  },
  {
    onStep: (result) => console.log(result.message)
  }
);

await agent.start();

Execution Modes

human-approved

Requires explicit approval for sensitive actions. Best for production-like workflows.

autonomous

Runs actions continuously with fewer pauses. Best for rapid iteration and demos.

Planner Options

heuristic

Zero-dependency regex-based planner. Works offline. Best for simple, predictable goals.

webllm

Delegates to a local WebLLM bridge (window.__browserAgentWebLLM). Fully private, no API calls.

page-agent

Delegates to an alibaba/page-agent bridge (window.__browserAgentPageAgent). Best for complex multi-step goals.

Recommended Integration Flow

  1. Create an agent with your goal and preferred mode.
  2. Attach callbacks (`onStep`, `onApprovalRequired`, `onDone`) to drive UI state.
  3. Call start() to begin planning and execution.
  4. If approval is required, surface the action and call resume().
  5. Use stop() for cancellation and reset.

Safety Notes

  • Prefer scoped selectors for deterministic action targeting.
  • Use human-approved mode for workflows that mutate critical data.
  • Log `onStep` output for auditability and debugging.
